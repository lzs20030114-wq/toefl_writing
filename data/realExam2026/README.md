# realExam2026 — 提取出来的 2026 改后真题(存档文件夹)

就是个**存放文件夹**,放从 `D:\桌面\【2026改后全科真题】` 提取出来的真题文本,供以后出题管线调用参考。不是正式题库,无需 manifest/接线。

- **来源**:2026 改后新托福机经(闲鱼包,2026-01~05),约 51 套不同卷
- **性质**:`recalled` 档(真实题目转写,非 ETS 逐字官方)——话题/结构/难度可信,精细措辞带回忆重构的保留意见
- **提取方式**:文字PDF→`pdftotext`;图片PDF→`scripts/ops/render-pdf.py`+`ocr-pdf.py`(本地RapidOCR,零token)
- **内容**:按题型分文件存(听力/阅读/写作/口语),每条带来源卷+日期
